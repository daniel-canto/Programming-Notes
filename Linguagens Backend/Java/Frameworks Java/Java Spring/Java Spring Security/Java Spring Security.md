### O que é?
É um módulo do ecossistema Java Spring responsável por lidar com a segurança em aplicações Java, principalmente quando se diz em **autenticação**, **autorização** e **proteção contra ataques** (como brute force, session fixation e etc).

Por padrão, esse módulo quando utilizado em um projeto, ele já torna todas as endpoints da API protegidas. Mas por padrão, a forma de autenticação utilizada é login e senha. Portanto, para APIs backend não é tão interessante assim. Se você jogar uma endpoint da sua API no navegador, o próprio Spring te redireciona para uma endpoint chamada "login", onde o o padrão default é "user" e uma senha gerada aleatoriamente no terminal ao executar a aplicação.
