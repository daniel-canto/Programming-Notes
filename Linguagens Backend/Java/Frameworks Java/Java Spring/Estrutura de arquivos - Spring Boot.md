Aqui será explicado os principais arquivos do Java Spring quando gerado com o Spring Boot, utilizando o Maven.

#### `pom.xml`
Este arquivo serve para listar as dependências do projeto que o Maven utilizará, identificando as dependências e suas versões.

#### `/.mvn/wrapper/maven-wrapper.properties
Aqui fica as configurações  do Maven.

#### `/src/test`
Dentro dessa pasta, fica os nossos testes unitários.

#### `/src/main/resources`
Nessa patas, colocamos arquivos de configurações, templates e arquivos estáticos, como script SQL, CSS, JS.  

###### `/resources/application.properties
Aqui é o arquivo onde fazemos algumas configurações específicas para o aplicativo.

#### `/src/main/java`
Aqui é onde irá conter a lógica da nossa aplicação.
